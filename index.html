How to redirect your website HTTP to HTTPS using .htaccess

3 July, 2021

You can redirect your website to a secure protocol by including only two lines of code in your .htaccess file.

# Redirect to HTTPS
RewriteCond %{HTTPS} off
RewriteRule (.*) https://%{HTTP_HOST}%{REQUEST_URI} [R,L]
